Full Stack Developer

Armonk, New York Full Time 6.5 LPA

Job description

A Full-Stack developer is a professional responsible for working on both front-end and back-end development processes.They design, develop, and maintain fully-fledged and functioning platforms with databases or servers.

Responsibility

The responsibilities of being a full stack developer can be daunting to manage in the beginning.The more you start knowing the technicality and gain fluency in FSD, the easier and more fun it will become.

  • Collaborate with development teams and product managers to create innovative software solutions.
  • Keeping up with the latest advancements in programming languages and server apps.
  • Design a plan for stability, scalability, performance optimization, and ongoing improvement.
  • The developer is in charge of creating APIs depending on the architecture of the production application.
  • Keep track of new development-related tools, frameworks, methods, and architectures.

Qualifications

  • HTML
  • CSS
  • JavaScript
  • DOM Manipulation
  • Java
  • OOPs
